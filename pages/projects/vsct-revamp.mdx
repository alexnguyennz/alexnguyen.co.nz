A revamp of a website using 11ty, CloudCannon, and Tailwind CSS. Originally built with WordPress and hosted on traditional Nginx.

Both 11ty and CloudCannon are the first static site generator and modern/headless content management system that I've used. Very happy with these choices I made for developing and implementing this site.

**Tools**
- <a href="https://www.11ty.dev/" target="_blank" rel="noreferrer noopener nofollow">11ty</a> - static site generator
- <a href="https://cloudcannon.com/" target="_blank" rel="noreferrer noopener nofollow">CloudCannon</a> - content management system, hosting
- <a href="https://tailwindcss.com/" target="_blank" rel="noreferrer noopener nofollow">Tailwind CSS</a> - CSS framework
- <a href="https://splidejs.com/" target="_blank" rel="noreferrer noopener nofollow">Splide</a> - image slider
- <a href="https://postcss.org/" target="_blank" rel="noreferrer noopener nofollow">PostCSS</a> - CSS JS processor
- <a href="https://github.com/artstorm/eleventy-plugin-seo" target="_blank" rel="noreferrer noopener nofollow">eleventy-plugin-seo</a> - Eleventy SEO plugin
- <a href="https://github.com/kangax/html-minifier" target="_blank" rel="noreferrer noopener nofollow">html-minifier</a> - HTML minifier
- <a href="https://github.com/markdown-it/markdown-it" target="_blank" rel="noreferrer noopener nofollow">markdown-it</a> - Markdown parser
