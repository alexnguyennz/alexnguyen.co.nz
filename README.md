My personal site at [alexnguyen.co.nz](https://alexnguyen.co.nz).

This website is mainly built with Next.js, and Tailwind for styling.

I've been away from web development for a few years and wanted to become familiar with modern tools so I thought a popular framework would be a good start - even though a pure static site generator would have been more suitable for this site. I've learned a lot by adding features this site doesn't necessarily need, like a contact form (when just an email address is probably easier when people can use their default preferred email client to send a basic message).

With the help of fantastic services like Netlify, I can keep this site running at no cost outside of domain fees.

**Tools**

- [Next.js](https://nextjs.org/) - web framework
- [Netlify](https://netlify.com/) - static site hosting
- [Tailwind CSS](https://tailwindcss.com/) - CSS framework
- [Cloudinary](https://cloudinary.com/) - media management
- [mds-jx](https://github.com/mdx-js/mdx) - Markdown processing
- [React Hook Form](https://react-hook-form.com/) - form validation
- [Framer Motion](https://www.framer.com/motion/) - library for page animation effects
- [next-seo](https://github.com/garmeeh/next-seo) - SEO management
